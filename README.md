# fitpulse_agent_platform
面向健康管理场景的智能体创作与运行平台，支持智能体、知识库配置。

FitPulse Agent Platform 是一个面向「健康管理 / 健康打卡 / 运动记录」场景的 智能体应用平台。
平台支持通过可视化方式完成 智能体配置、知识库管理和工作流编排：

- 智能体（Agent）：支持多角色、多工具的智能体配置，可接入插件与外部 API，实现个性化健康问答、计划制定与数据解析。

- 知识库（Knowledge Base）：支持上传和管理健康相关文档、FAQ 与结构化数据，为智能体提供可检索的专业知识支撑。

- 工作流（Workflow）待开发：提供可视化工作流引擎，将模型调用、插件调用、数据处理和消息推送串联起来，实现健康评估、提醒与报告生成等自动化流程。
